# lutece-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Advanced

You can run scripts with additional features using the GUI.

### Install Vue CLI
```
npm install -g @vue/cli
```

### Run Vue CLI GUI
```
vue ui
```
